TROUBLESHOOTING
========================================

Problem: "API Error" or "Network Error"
Fix: Check API URL in environment variables
     ‚Üí Should point to: https://your-render-app.onrender.com/api/v1

Problem: Calendar still shows wrong dates
Fix: Hard refresh browser (Ctrl + Shift + R)
     ‚Üí Clear browser cache completely

Problem: 404 errors on navigation
Fix: SiteGround needs .htaccess for React Router
     ‚Üí Create .htaccess in public_html:
     
     <IfModule mod_rewrite.c>
       RewriteEngine On
       RewriteBase /
       RewriteRule ^index\.html$ - [L]
       RewriteCond %{REQUEST_FILENAME} !-f
       RewriteCond %{REQUEST_FILENAME} !-d
       RewriteCond %{REQUEST_FILENAME} !-l
       RewriteRule . /index.html [L]
     </IfModule>

Fix: Check browser console (F12)
